#Linux #Python

linux中yum安装python

添加源

首先搜索python源

yum search python

查看yum源中是否有python

如图,没有请添加阿里云源
python36就是python3.6版本
我们使用yum来安装python

yum install -y python36

检查版本

python -V

结果是这样的默认为自带Python 2.7.5

[root@ixiao9 ~]# python -V
Python 2.7.5

我们可以使用python3进入python3环境

[root@ixiao9 ~]# python3
Python 3.6.6 (default, Mar 29 2019, 00:03:27) 
[GCC 4.8.5 20150623 (Red Hat 4.8.5-36)] on linux
